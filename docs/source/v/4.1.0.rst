v4.1.0
==========

Added
----------
* ``TextureArrayDecoder``: new decoder for ``.texarr`` archives.
* ``TextureArrayEncoder``: new encoder to ``.zip``.
* ``convert.texarr_to_zip``: new converter.
* Templates: launcher torrent binary template.

Changed
----------
* ``GlbEncoder``: single bind matrix.
* ``GlbEncoder``: position accessor bounds.
* ``scfile.formats``: package imports improved.

Fixed
----------
* ``GlbEncoder``: handle meshes without skinning links when skeleton present.

Optimized
----------
* ``StructIO``: read array optimization (``.mcsa`` parsing 53ms → 8ms).
